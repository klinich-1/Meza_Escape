# Maze Escape

Maze Escape — это простая игра на Python с библиотекой PyGame, в которой игрок должен найти выход из лабиринта, собрав кристаллы и избегая охотника.

## Установка

Рекомендуется создать виртуальное окружение и установить зависимости из `requirements.txt`:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

## Запуск

```powershell
python -m maze_escape
```

## Управление

- `WASD` или стрелки: движение.
- `Space`: телепорт-миг к безопасной видимой клетке.
- `R`: начать новую игру.
- `P`: пауза.
- `Esc`: выход.

## Сложность

При старте игры в меню выбирайте уровень сложности:
- `Легко` — больше проходов, медленнее охотник, большая зона видимости.
- `Средне` — стандартные параметры.
- `Сложно` — меньше проходов, более быстрый охотник, меньшее поле зрения.

Если вы выиграли на уровне `Легко`, следующий рестарт переводит игру на `Средне`. Победа на `Средне` переводит на `Сложно`.

## Тестирование

Для запуска модульных тестов используйте стандартный `unittest`:

```powershell
python -m unittest discover -s tests
```

Для проверки алгоритмов и логики используйте модульные тесты из каталога `tests`.

## О проекте

Игра разделена по принципам Model-View:

- `game_model.py` содержит игровую логику и состояния.
- `app.py` отвечает за интерфейс, отрисовку и ввод.
- `algorithms.py` реализует генерацию лабиринта, поиск пути и видимость.

## Алгоритмы

- `generate_maze()`: генерация лабиринта на основе случайного обхода DFS.
- `astar()`: поиск пути охотника к игроку.
- `raycast_fov()`: расчет поля зрения для эффекта тумана войны.
